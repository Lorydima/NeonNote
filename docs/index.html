NeonNote

NeonNote is a simple note-taking application
with a user-friendly interface

Developed by LDM Dev

Developed in Python

NeonNote is developed using:
Tkinter Library for GUI and
filedialog function. For Advanced
function is used the libraries: pyperclip, time,
os and webbrowser.

Large Text Box


NeonNote has a large text input area that allows
for a high-quality view of the text being written.

Advanced Functions

  • Change Font in arial, calibri or segoe UI
  • Write actual date and time
  • A count chars/words function
  • Shortcut keys for copy, paste, save and opens files
