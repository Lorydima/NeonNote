# NeonNote V1.1 - Requirements
# All dependencies for NeonNote V1.1 application
#
# Note: NeonNote primarily uses the Python Standard Library.
# No external packages are strictly required to run the core app.
#
# Python Version: 3.x (3.8 or higher recommended)
#
# To install Python dependencies (if any):
# pip install -r requirements.txt
#
# Standard Library modules used (no installation required):
# - tkinter (GUI Framework)
# - time (time/date helpers)
# - os (Operating System interface)
# - sys (System-specific parameters)
# - json (JSON file handling)
# - webbrowser (Web browser controller)
# - tempfile (temporary files)
# - re (regular expressions)
#
# Optional runtime dependencies (install only if you need the feature):
# - reportlab   -> used by the Export PDF feature
# - pywin32     -> optional Windows printing helpers (win32api / win32print)
# - pyperclip   -> clipboard copy/paste support
#
# Optional build tool (to create a Windows EXE):
# - pyinstaller>=5.0
#
# Recommended minimal install (uncomment to use):
# reportlab
# pywin32
# pyperclip
#
# Optional build-time tool (uncomment to use when packaging):
# pyinstaller>=5.0
